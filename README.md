# 安全科学计算器网页版

这是从 `calculator_security_package.zip` 中的安全计算器逻辑整理出的静态网页版，可直接发布到 GitHub Pages。

## 本地打开

直接打开 `index.html` 即可使用。

## 支持

- 四则运算、取模、幂运算
- 函数：`sqrt`、`sin`、`cos`、`tan`、`log`、`log10`、`exp`、`abs`、`round`、`fact`
- 常量：`pi`、`e`
- 上一次结果：`ans`
- 表达式长度、token 数量、指数、阶乘参数和结果大小限制
- 手机端点表达式区域不会弹出系统键盘，使用页面内置按键输入
- 删除键支持一键删除刚插入的函数片段，例如 `cos(`、`sqrt(`、`log10(`
- 页面脚本带版本号，手机浏览器会自动获取最新功能代码
- 移动端优化：禁止双击缩放、长按菜单、文本选中、点击高亮和页面橡皮筋回弹；按键区限制同一时间只响应一个触点
- 支持轻微震动反馈；不支持震动 API 的浏览器会自动忽略

网页版本使用 JavaScript 自带的实数数学函数，不支持 Python 版里的复数结果。
